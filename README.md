# Block Explorer
A blockchain explorer application which will probably be deprecated and abandoned the first chance I get.

## Running The App
If you aren't sure which platform you want to use initially, run the following command:
```bash
npx expo start
```

### Android
```bash
npx expo start --android
```

### iOS
```bash
npx expo start --ios
```

## Web
```bash
npx expo start --web
```
